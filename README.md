# MTG-Moxfield-Deck-Downloader

This is a simple program that downloads commander Magic The Gathering decks from Moxfield. <br /> <br />
It will download each card and token in the deck as jpg images, alongside renaming each image to be the card name. <br /> <br />

# How To Use
The only thing needed is to input the deck url into the program (The deck must be public or unlisted on Moxfield). <br /> <br />
After clicking the 'download deck' the program will download the cards, once it is done it will tell you if it was successful or not. <br /> <br />
If it succeeded it will auto open the folder that cards were downloaded to. <br /> <br />
The cards are separated into four folders, commander, cards, tokens, and backs (Backs being the backside of double faced cards/tokens). <br /> <br />
